# Genius歌词下载器 - 多任务专业版

## 项目概述

Genius歌词下载器多任务专业版是一个基于Python Tkinter开发的图形用户界面工具，专门用于从Genius.com网站批量下载歌词。该版本在原有单任务版本的基础上，引入了多任务管理功能，允许用户同时运行多个独立的歌词下载任务。

## 核心功能

### 1. 多任务管理
- **独立任务选项卡**：每个任务都在单独的标签页中运行，互不干扰
- **任务列表面板**：左侧面板显示所有任务的状态概览
- **任务切换**：通过点击任务列表或标签页轻松切换不同任务
- **任务状态监控**：实时显示每个任务的运行状态和进度

### 2. 任务管理操作
- **新建任务**：创建自定义名称的下载任务
- **重命名任务**：随时修改任务名称以便于识别
- **删除任务**：移除不再需要的任务（不会删除已下载文件）
- **任务配置**：每个任务可独立设置API密钥和保存路径

### 3. 智能特性
- **断点续传**：支持任务中断后的恢复功能
- **设置自动保存**：每个任务的配置和进度自动保存
- **API密钥管理**：多任务共享API密钥池，智能调度请求
- **独立配置文件**：每个任务使用独立的设置和断点文件

## 系统要求

- Python 3.6+
- 依赖库：
  - tkinter（通常随Python一起安装）
  - 其他依赖项见`requirements.txt`

## 安装步骤

1. 克隆或下载项目文件
2. 确保已安装单任务版本的`Genius_Lyrics_Crawl.py`
3. 如果可用，安装`rate_limiter.py`模块以启用API速率限制功能
4. 运行主程序：
   ```bash
   python Genius_Lyrics_Crawl_MultiTask.py
   ```

## 使用方法

### 1. 启动程序
运行主程序后，将看到主界面分为两个主要区域：
- 左侧：任务列表和状态面板
- 右侧：当前选中任务的操作界面

### 2. 创建新任务
1. 点击"➕ 新建任务"按钮
2. 输入任务名称（必填）
3. 可选的API密钥（可在任务中单独设置）
4. 可选的保存路径（默认：桌面/Genius歌词）

### 3. 配置任务
每个任务标签页内包含完整的单任务功能：
- 输入艺人名称或URL
- 配置下载选项
- 设置API密钥
- 选择保存目录

### 4. 管理任务
- **切换任务**：点击左侧任务列表或顶部标签页
- **查看状态**：左侧面板显示选中任务的详细状态
- **批量操作**：可同时启动多个任务进行并行下载

### 5. 保存与恢复
- **自动保存**：程序关闭时自动保存所有任务状态
- **手动保存**：每个任务可独立保存设置
- **恢复功能**：重新打开程序时自动加载上次的任务状态

## 文件结构

```
multi_task_manager.py      # 主程序（多任务管理器）
Genius_Lyrics_Crawl.py    # 单任务核心功能模块
rate_limiter.py           # （可选）API速率限制器

# 配置文件（自动生成）
multi_task_config.json    # 多任务管理器配置
lyrics_downloader_task_[任务名].json    # 每个任务的独立配置
lyrics_downloader_resume_[任务名].json  # 每个任务的断点信息
```

## 配置说明

### 1. API密钥设置
- 每个任务可以设置独立的Genius API密钥
- 如果启用rate_limiter，所有密钥将被集中管理以提高效率
- 可在创建任务时设置，也可在任务界面中单独配置

### 2. 保存路径
- 每个任务可指定独立的保存目录
- 默认保存在用户的桌面"Genius歌词"文件夹
- 支持嵌套文件夹按艺人分类

### 3. 任务状态持久化
- 任务名称、状态、进度等信息自动保存
- 独立的配置和断点文件避免冲突
- 支持程序意外关闭后的状态恢复

## 使用技巧

1. **并行下载**：创建多个任务同时下载不同艺人的歌词
2. **分类管理**：按语种、流派或地区创建不同任务
3. **优先级调度**：重要的任务先开始，后台任务持续运行
4. **资源管理**：监控每个任务的进度和状态，合理分配系统资源
5. **断点利用**：网络中断后可恢复下载，无需重新开始

## 注意事项

1. **API限制**：请遵守Genius API的使用条款和频率限制
2. **文件保存**：删除任务不会删除已下载的文件，文件需手动管理
3. **网络要求**：稳定的网络连接对批量下载很重要
4. **存储空间**：确保有足够的磁盘空间保存歌词文件
5. **备份建议**：定期备份重要的配置文件

## 故障排除

### 常见问题
1. **无法启动任务**：检查API密钥是否有效
2. **下载中断**：检查网络连接，使用断点续传功能恢复
3. **界面卡顿**：同时运行的任务过多，建议暂停部分任务
4. **配置丢失**：检查是否有写入权限，配置文件是否被误删

### 技术支持
- 确保所有依赖模块已正确安装
- 检查Python版本兼容性
- 查看控制台输出获取详细错误信息
